## Installation

````
pod 'DartleSDK', :git => 'git@bitbucket.org:nfnty_admin/dartlesdk_ios.git'
````

* Run `pod install` first.

## Getting Started

````
#import "DartleManager.h"    // import all the things
````

## Dependencies

* [AFNetworking]
* [PromiseKit]
* [PromiseKit+AFNetworking]

## Requirements

* iOS 7.0+
* ARC
